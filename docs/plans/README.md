# Plans

This folder contains short-lived planning handoffs for active Bondstone work.
Plans should not become a parallel durable documentation system.

Use plans to group proposed ADRs, backlog themes, and near-term sequencing.
When a plan item becomes durable behavior, move the decision into an accepted
ADR and stable docs. When a plan item becomes implementation work, track it in
GitHub Issues or GitHub Projects using [../github-workflow.md](../github-workflow.md).
