# Agent Skills

This folder contains repository-local agent skills for Bondstone maintainers.

See [AGENTS.md](AGENTS.md) for the skill authoring rules and
[../../docs/adr/README.md](../../docs/adr/README.md) for ADR workflow rules
used by the current skills.

## Skills

- `bondstone-adr-create`: create a new ADR and apply accepted decisions.
- `bondstone-adr-update`: update an existing ADR without hiding history.
- `bondstone-adr-supersede`: replace an old ADR with a new decision.
- `bondstone-adr-archive`: archive or remove obsolete ADR material.
- `bondstone-github-issue-workflow`: create, triage, prioritize, work, and
  complete GitHub Issues and Project items.
