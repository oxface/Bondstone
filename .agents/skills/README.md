# Agent Skills

This folder contains repository-local agent skills for Bondstone maintainers.

See [AGENTS.md](AGENTS.md) for the skill authoring rules. Durable
requirements, architecture, implementation sequencing, and lean agent
guardrails are owned by SpecKit memory and stable repository docs.

## Skills

- `bondstone-github-issue-workflow`: create, triage, prioritize, work, and
  complete GitHub Issues and Project items.
