# Pre-Restart ADR Archive

Date: 2026-06-16

This folder preserves Bondstone's original ADR sequence from the first public
MVP and post-MVP architecture pivot.

The files in this archive are retained for traceability. They are not the
active decision navigation path. Current decisions live in the active ADR set
under `docs/adr/`.

Use this archive when older context is needed, especially for decisions around
removed packages, Rebus and Service Bus exploration, direct PostgreSQL
persistence, topology diagnostics, capability pipelines, and the path toward
the current simplified architecture.

Do not update archived ADRs to describe current behavior. Update or supersede
the active ADRs instead, then apply the current rule into stable docs.
