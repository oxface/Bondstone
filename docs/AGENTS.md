# Documentation Agent Index

This folder contains stable docs, ADRs, backlog material, and archive material.

Start with:

- [README.md](README.md) for the documentation model and ownership map.
- [adr/README.md](adr/README.md) before changing ADRs.
- [architecture/README.md](architecture/README.md) before changing runtime
  architecture docs.
- [backlog/README.md](backlog/README.md) before changing ad hoc planning
  notes.
- [repository.md](repository.md) before changing repository layout, tooling, or
  context-index structure.

Keep stable docs as the current operating contract. Keep ADRs as the decision
trail. Use local README and AGENTS files as indexes into those docs rather than
as duplicate rule stores.
