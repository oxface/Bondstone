# Backlog Agent Index

This folder contains ad hoc planning notes, not a maintained implementation
roadmap.

Start with:

- [README.md](README.md) for the planning model and active issue notes.
- [00-plans.md](00-plans.md) for loose known pressure points.
- [../adr/README.md](../adr/README.md) before turning planning decisions into
  ADRs.
- Stable docs under [../architecture](../architecture) before applying
  accepted behavior.

Use active issue notes only when they match the current task. When an issue is
resolved, update ADRs and stable docs, return remaining ideas to
`00-plans.md`, and remove the resolved issue note unless it still contains
short-term context that has not moved elsewhere.
