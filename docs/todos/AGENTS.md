# Planning Notes Agent Index

This folder contains transitional planning notes.

Start with:

- [README.md](README.md) for the folder purpose and note index.
- [../github-workflow.md](../github-workflow.md) before converting plan items
  into GitHub Issues or Project items.
- [../adr/README.md](../adr/README.md) before turning a plan item into an ADR.
- [../architecture/README.md](../architecture/README.md) before changing
  runtime architecture docs.

Do not treat planning notes as stable operating contracts. When a plan item is
implemented, update the relevant stable docs and ADRs instead of leaving the
plan as the only source of truth.
